Step 1: Install Node.js and npm

üîß Installing Node.js

Node.js is a JavaScript runtime that allows us to run JavaScript outside the browser. It comes with npm (Node Package Manager), which we'll use to manage our project dependencies.

  1. Visit nodejs.org
  2. Download the LTS (Long Term Support) version
  3. Run the installer and follow the setup wizard
  4. Verify the installation by opening your terminal and running:
$ node --version v18.17.0 $ npm --version 9.6.7
‚ö†Ô∏è Troubleshooting: If the commands above don't work, you may need to restart your terminal or computer after installation. On Windows, make sure to run the terminal as Administrator if you encounter permission issues.

Step 3: Create Your First React Application

üöÄ Using Create React App

Create React App is the official tool for setting up new React projects. It configures everything you need to start building React applications.

Open your terminal and run the following commands:

$ npx create-react-app my-first-react-app $ cd my-first-react-app $ npm start
üí° What's happening here?
  • npx runs the latest version of create-react-app without installing it globally
  • create-react-app my-first-react-app creates a new React project in a folder called "my-first-react-app"
  • cd my-first-react-app navigates into the project folder
  • npm start starts the development server

Step 4: Understanding the Project Structure

After creating your React app, you'll see this folder structure:

my-first-react-app/ ‚îú‚îÄ‚îÄ public/ ‚îÇ ‚îú‚îÄ‚îÄ index.html # Main HTML file ‚îÇ ‚îú‚îÄ‚îÄ favicon.ico # Website icon ‚îÇ ‚îî‚îÄ‚îÄ manifest.json # PWA configuration ‚îú‚îÄ‚îÄ src/ ‚îÇ ‚îú‚îÄ‚îÄ App.js # Main App component ‚îÇ ‚îú‚îÄ‚îÄ App.css # App styles ‚îÇ ‚îú‚îÄ‚îÄ index.js # Entry point ‚îÇ ‚îú‚îÄ‚îÄ index.css # Global styles ‚îÇ ‚îî‚îÄ‚îÄ App.test.js # Test file ‚îú‚îÄ‚îÄ package.json # Project dependencies ‚îî‚îÄ‚îÄ README.md # Project documentation

Key Files Explained

  • public/index.html: The single HTML page that loads your React app
  • src/index.js: The JavaScript entry point that renders your app
  • src/App.js: Your main React component
  • package.json: Lists all dependencies and scripts

Step 5: Your First Success!

If everything went well, you should see:

  • Your terminal showing "Compiled successfully!"
  • Your browser automatically opening to http://localhost:3000
  • The React logo spinning on the page
üéâ Congratulations! You've successfully set up your React development environment and created your first React application. The development server is now running and will automatically reload when you make changes to your code.

Development Server Commands

$ npm start # Start development server $ npm test # Run tests $ npm run build # Build for production $ npm run eject # Eject from Create React App (advanced)
‚ö†Ô∏è Important: Don't run npm run eject unless you really know what you're doing. It's a one-way operation that exposes all the configuration files, making your project more complex to manage.

Troubleshooting Common Issues

  • Port 3000 already in use: The server will automatically try port 3001, 3002, etc.
  • Permission errors: Try running your terminal as administrator (Windows) or using sudo (Mac/Linux)
  • Slow installation: This is normal for the first time; Create React App downloads many dependencies
